# 快速上手

这篇文档将指引如何快手上手使用组件库。

### Vue 版本
建议使用 Vue.js >= `3.3.2`

### 安装
你可以从 `npm` 或 `yarn` 安装
```Shell
# npm
npm i birdpaper-ui

# yarn
yarn add birdpaper-ui
```

### 全量引入
```JS
import App from "./App.vue"

import BirdpaperUI from "birdpaper-ui"
import "birdpaper-ui/dist/index.css"

const app = createApp(App)
app.use(BirdpaperUI)
app.mount("#app")
```
